= Topic EIP
:doctitle: Topic
:shortname: topic
:description: Topic which sends to all destinations.
:since: 
:supportlevel: Stable

Topic mode for the xref:loadBalance-eip.adoc[Load Balancer] EIP.
With this policy the exchange will be sent to all endpoints (or processors).

== Examples

In this example we send the message to all three endpoints:

[source,java]
----
from("direct:start")
    .loadBalance().topic()
        .to("seda:x")
        .to("seda:y")
        .to("seda:z")
    .end();
----

In XML you'll have a route like this:

[source,xml]
----
<route>
<from uri="direct:start"/>
    <loadBalance>
       <topic/>
       <to uri="seda:x"/>      
       <to uri="seda:y"/>      
       <to uri="seda:z"/>       
    </loadBalance> 
</route>
----
